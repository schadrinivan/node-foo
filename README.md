# node-foo

A simple Node.js server with an Express endpoint.

## Installation

```bash
npm install
```

## Usage

Start the server:

```bash
npm start
```

The server will run on port 3000 by default.

## Endpoints

### GET /hi

Returns "Hi There"

**Example:**

```bash
curl http://localhost:3000/hi
```

**Response:**

```
Hi There
```
